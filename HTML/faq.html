Preguntas Frecuentes (FAQ)

¿Qué sistemas de rol se pueden usar, puedo usar pathfinder por ejemplo?

Esta página esta enfocada exclusivamente para la creación de personajes aleatoreos de Dungeons and Dragons 5.5 (anteriormente conocida como 2024).

¿Qué versión de D&D se pueden usar?

Esta página esta enfocada exclusivamente en la versión de Dungeons and Dragons 5.5 (anteriormente conocida como 2024).

¿Cómo puedo añadir mis propios manuales o reglas caseras?

Podés arrastrar o seleccionar cualquier archivo en formato `.csv` usando el botón dashed de carga. El sistema absorberá las opciones y las sumará al pool de generación inmediatamente.

¿Qué formato estricto debe tener mi archivo CSV personalizado?

Debe incluir una fila de cabecera con las columnas: Tipo, Nombre, Dependencia. Los tipos válidos que el motor reconoce son: Clase, Subclase, Raza, Subraza y Trasfondo.

¿Cómo calcula el sistema la distribución de dotes?

Utiliza el bloque oficial 'Standard Array' de D&D 2024 (15, 14, 13, 12, 10, 8). El algoritmo identifica las estadísticas primarias de la clase obtenida y les asigna los valores más altos de forma fija, distribuyendo el resto aleatoriamente para garantizar que el personaje sea completamente jugable.

¿Por qué se iluminan dos estadísticas con un borde rojo brillante al generar un personaje?

Indica que esas dos dotes recibieron los aumentos automáticos de tu Trasfondo (+2 y +1). El algoritmo analiza de forma inteligente las opciones válidas que otorga tu Trasfondo y las inyecta en tus estadísticas base más altas para optimizar mecánicamente la build del personaje.

¿Por qué a veces la sección de "Subclase" o "Subraza" me dice "No aplica"?

El generador sigue estrictamente las reglas oficiales de progresión de D&D 2024. Las subclases se eligen de forma universal a partir de Nivel 3, por lo que si generás un personaje de nivel 1 o 2, la sección se ocultará automáticamente. Con las subrazas ocurre igual: si la Raza sorteada no cuenta con variantes en los manuales activos, el sistema indicará que no aplica.

Al desactivar un libro o borrarlo, ¿se pierde el personaje que ya tengo en pantalla?

No. El panel de manuales y los chips manejan exclusivamente la memoria global de opciones disponibles para el sorteo. Si desactivás un libro o eliminás un CSV personalizado, el personaje que tenés visible se mantendrá intacto, pero sus datos ya no se tendrán en cuenta para las próximas generaciones.

¿Puedo cargar varios archivos CSV personalizados al mismo tiempo?

¡Sí, totalmente! El cargador de archivos soporta selección múltiple. Podés seleccionar o arrastrar una tanda de varios manuales caseros juntos y el motor asíncrono los procesará e incorporará en paralelo al pool de datos inmediatamente.
